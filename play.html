Instructions:
Choose a name and press Enter to start the game.
Arrow keys and ASDW to play and Z or M to dash.
Notes:
Credits are in game.

Flash warning! This game has a set of flashing lights that may cause eleptic seizures. If you are diagnosed with those conditions, please refrain from using this game.

Violence warning! This game has several scenes including blood and/or gore. If you wish to have a less, bloody version, please go to settings and change the setting called "Violence". Note! This will not remove all blood scenes, but will only allow neccessary violent scenes.
